## Technology Stack

### Frontend

- **Framework:** React 19.2.3
- **Language:** TypeScript 5.8
- **Styling:** Tailwind CSS 4
- **Bundler:** Vite
- **Testing:** React Testing Library + Jest

### Backend

- **Runtime:** Node.js
- **Framework:** Express.js
- **Language:** TypeScript
- **Database:** Supabase (PostgreSQL)
- **API:** REST with real-time subscriptions

### OCR & AI

- **Primary Model:** Gemini 2.5 Flash (Google)
- **Fallback Model:** SiliconFlow Qwen 2.5 VL (72B)
- **Processing:** < 5 seconds per document
- **Accuracy Target:** 95%+ for invoices

### Accounting Integration

- **German Standards:** SKR03 (standard accounting plan)
- **Tax System:** ELSTER export support
- **Software:** DATEV export integration
- **Multi-Language:** German, English, French support

### Infrastructure

- **Hosting:** Cloud deployment (Supabase + Vercel)
- **Monitoring:** Prometheus + Grafana
- **Logging:** ELK Stack
- **CI/CD:** GitHub Actions
- **Containerization:** Docker
