# Story K1 — Feldkatalog + Datenmodell-Abgleich

## Beschreibung

Als Nutzer:in/Steuerberater:in möchte ich, dass alle relevanten Belegfelder klar definiert und konsistent im Datenmodell vorhanden sind, damit Exporte vollständig und nachvollziehbar sind.

## Akzeptanzkriterien

- Es existiert eine eindeutige Liste der zu erfassenden Felder (Belegkopf + Zahlung + organisatorisch + Flags).
- Für jedes Feld ist dokumentiert:
  - Quelle (OCR/Manuell/abgeleitet)
  - Datentyp/Format (z.B. ISO-Datum)
  - Optional/Pflicht (inkl. Bedingungen)
- Positionen werden als 1:n-Daten festgelegt (Line Items).
