# Story K2 — CSV/SQL Export erweitern (fehlende Spalten + Positionen)

## Beschreibung

Als Nutzer:in möchte ich Exporte erzeugen können, die alle relevanten Belegfelder enthalten (inkl. Zahlungsdaten) und Positionen separat exportieren, damit die Weiterverarbeitung (Steuerberater/Controlling) zuverlässig funktioniert.

## Akzeptanzkriterien

- CSV-Export `Belege` enthält mindestens zusätzlich:
  - `zahlungsmethode`
  - `zahlungsDatum` (ISO)
  - `zahlungsStatus`
  - `aufbewahrungsOrt`
  - `rechnungsEmpfaenger`
  - sowie weitere im Feldkatalog definierte Felder
- Zusätzlich wird eine zweite CSV exportiert:
  - `Positionen` mit Spalten: `doc_id`, `line_index`, `description`, `amount`
- SQL-Export wird um die neuen Felder erweitert (Schema konsistent zu Export-CSV und Datenmodell).
